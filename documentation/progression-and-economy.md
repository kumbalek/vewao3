# Progression & Economy

## Two Layers of Economy

The game operates two distinct economic systems:

1. **In-Run Economy** — Scrap and Components earned mid-mission, spent at shops and shipyards. Resets on death. Carried forward on safe return (with ship).
2. **Meta Economy** — Scrap Metal, Data Cores, and Void Crystals carried back from successful missions. Used to upgrade the permanent station.

**Ship state as a resource**: On safe return, your ship (deck + installed modules) survives intact. On death, the ship is lost. A well-built engine is valuable — risking it for deeper rewards is the core return decision, not a system mechanic.

---

## In-Run Economy

### Currency: Scrap (in-run)
Drops from every combat. Used at Shops and Shipyards during the mission.

| Source | Amount |
|--------|--------|
| Normal Combat win | 10–25 Scrap |
| Elite Combat win | 25–45 Scrap |
| Boss Combat win | 40–70 Scrap |
| Event rewards | 15–50 Scrap (varies) |
| Selling a module at Shipyard | 30–60% of install cost |

**Spending Scrap in-run**:
- Shop: Cards (15–50 Scrap), Modules (40–80 Scrap), Card removal (40 Scrap flat)
- Shipyard: Module installation (20–50 Scrap), Module upgrade (30 Scrap), Grid rearrangement (free)
- Events: Some risk-avoidance options cost Scrap

### Currency: Component Parts (in-run)
Rarer drop from Elites and events. Used exclusively at Shipyard nodes.

| Source | Amount |
|--------|--------|
| Elite Combat win | 1–2 Components |
| Special events | 1–3 Components |
| Specific modules (Scrap Compressor) | 1 per fire |

**Spending Components at Shipyard**:
- Install rare module tier (requires 1 Component)
- Unlock additional grid slot (requires 2 Components)
- Perform "deep upgrade" on module (beyond `+` tier, requires 2 Components)

## Meta Economy

### Currency: Scrap Metal (meta)
The most common meta currency. Accumulated across missions.

| Source | Amount |
|--------|--------|
| Successful mission return (per act cleared) | 30/50/80 Scrap Metal per act |
| Bonus for clearing boss | +40 Scrap Metal |
| Survival score bonus | 5–20 additional |
| Death penalty | Lose ALL in-run Scrap; keep Scrap Metal earned before death |

### Currency: Data Cores (meta)
Uncommon. Requires reaching Act 2+.

| Source | Amount |
|--------|--------|
| Act 2 boss kill | 1 Data Core |
| Act 3 boss kill | 2 Data Cores |
| Select events ("Analyze the transmission") | 1 Data Core |
| Research Lab (station module) — passive generation | 1 per 3 missions |

### Currency: Void Crystals (meta)
Rare. Endgame unlock gating.

| Source | Amount |
|--------|--------|
| Act 3 boss kill | 1 Void Crystal |
| Secret boss kill | 3 Void Crystals |
| Void Anomaly event | 1 Void Crystal |
| Ascension milestone completion | 1 Void Crystal |

### Death & Meta Currency
- On death: all in-run Scrap lost. Meta currencies earned before the run remain.
- Scrap Metal is NOT earned from a run that ends in death (incentive to return alive)
- Data Cores and Void Crystals found mid-run ARE kept on death (high-value items)
- Mind-Save Pod upgrades can preserve a % of in-run Scrap on death:
  - Tier 0: 0% preserved
  - Tier 1: 10% preserved
  - Tier 2: 25% preserved

## Unlock Progression

### Ships
Ships are unlocked by completing specific challenges (not just buying them):

| Ship | Unlock Condition |
|------|----------------|
| The Drifter | Starting ship (always available) |
| The Vanguard | Starting ship (always available) |
| The Weaver | Starting ship (always available) |
| The Bastion | 3 consecutive safe returns without dying |
| The Phantom | Defeat the Act 2 boss for the first time |
| The Foundry | Build 10 total Station modules |
| The Remnant | Discover (encounter in any run) 20 unique rare cards |
| The Convergence | Complete a full 3-act mission (reach and defeat Act 3 boss) |

### Card Pool Expansion
Cards are divided by **tier of availability**:

- **Core Pool**: Always available. The 10 cards per energy type that form the base game.
- **Extended Pool**: Unlocks after surviving to Act 2 for the first time. Uncommon cards enter pools.
- **Rare Pool**: Cards must be **discovered** (seen in a reward or shop) before they enter future shop rotations. First time you see a rare card, it "registers" and appears more frequently in future runs.
- **Signature Cards**: Ship-specific. Always available on that ship. Cannot appear on other ships.

### Module Pool Expansion
Same logic as cards: Core modules always available; Rare modules must be discovered first.

### Cosmetics
Cosmetics are vanity rewards — different frame styles for ships, card back designs, Stead skins. Earned through:
- Ascension milestones
- Achievement completions
- Challenge run clears
- (Future: premium purchase)

## Run Scoring

At the end of a completed run, a score is calculated for the leaderboard and personal bests:

| Component | Points |
|-----------|--------|
| Acts cleared | 500 per act |
| Elite enemies killed | 75 each |
| Normal enemies killed | 20 each |
| Boss kills | 200 per boss |
| Cards upgraded | 30 each |
| Modules upgraded | 50 each |
| Scrap returned | 1 per 5 Scrap |
| Ascension level | ×(1 + 0.15 per Ascension level) multiplier |
| Deaths in run | −200 per death (future: "no death" bonus run type) |

**Leaderboard scopes**: All-time personal best (per ship), Daily Challenge global leaderboard, Weekly Challenge leaderboard.

## Balance Philosophy

**Early runs should feel progressively unlocking** — the first few missions teach mechanics through limited options. As more modules and cards unlock, build variety explodes.

**The station economy should always feel meaningful** — there's never a point where you have "too much" Scrap Metal and nothing to spend it on. Later station modules are expensive, and cosmetics provide optional sinks.

**Void Crystals are rare enough to matter** — 1–2 per run if you push to Act 3. Endgame unlocks cost 3–8 Crystals. A new player won't unlock everything for many sessions.

**Scarcity drives decisions** — you rarely have enough in-run Scrap to buy everything at a shop. Choosing between a key card and a card removal (or module upgrade) is the core shop tension.

**Deeper is better, not just harder** — Act 2 and Act 3 rewards are substantially better than Act 1 grinding. Story content, narrative chains, and rare event pools all gate to later acts. A player who repeats Act 1 is leaving the interesting parts of the game unplayed — not just earning less, but seeing less.
